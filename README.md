# 🎯 Equitable ATS - Phase 1

AI-powered applicant tracking system with role-based access, job posting, and application tracking.

## ⚡ Quick Start

```bash
# Frontend (Terminal 1)
cd frontend
npm install
npm run dev

# Backend (Terminal 2)
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Visit: http://localhost:5174
